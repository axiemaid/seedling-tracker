# Seedling Growth Tracker

Track 28 hydroponic seedlings with an ESP32-WROVER-CAM and a Mac Mini.

## Architecture

```
ESP32-WROVER-CAM (overhead)        Mac Mini
┌─────────────────────┐           ┌──────────────────────────┐
│  Serves JPEG via    │  WiFi     │  tracker.py              │
│  HTTP at /capture   │ ◄───────► │  - Fetches JPEG          │
│                     │           │  - Perspective correction │
│  Endpoints:         │           │  - HSV green segmentation │
│    /capture  (jpg)  │           │  - Per-cell measurement   │
│    /stream   (mjpeg)│           │  - CSV logging            │
│    /status   (json) │           │  - Growth charts          │
└─────────────────────┘           └──────────────────────────┘
```

## Hardware

- **ESP32-WROVER-CAM** (with OV2640 sensor)
- **Mac Mini** (any recent model)
- USB power supply for ESP32
- Mounting: position camera directly above the seedling tray
- Recommended: 30-50cm above seedlings for full tray coverage

## Setup

### 1. ESP32-CAM Firmware

**Requirements:**
- Arduino IDE (or PlatformIO)
- ESP32 board package installed

**Steps:**
1. Open `esp32-cam/seedling_cam/seedling_cam.ino` in Arduino IDE
2. Edit WiFi credentials:
   ```c
   const char* ssid     = "YOUR_WIFI_SSID";
   const char* password = "YOUR_WIFI_PASSWORD";
   ```
3. Select board: **AI Thinker ESP32-CAM** (or your specific WROVER variant)
4. Select partition scheme: **Huge APP (3MB No OTA/1MB SPIFFS)**
5. Upload (hold BOOT button during upload if needed)
6. Open Serial Monitor at 115200 — note the IP address printed on boot
7. Test: open `http://<IP>/capture` in a browser — you should see a photo

**Tip:** Assign a static IP to the ESP32 on your router so it doesn't change.

### 2. Mac Mini Software

**Requirements:**
```bash
pip3 install opencv-python numpy matplotlib
```

**Steps:**
1. Navigate to the `mac-mini/` directory
2. Run calibration:
   ```bash
   python3 tracker.py --calibrate
   ```
3. Follow the prompts:
   - Enter your ESP32's IP address
   - Click the 4 corners of your seedling grid (top-left → top-right → bottom-right → bottom-left)
   - Tune HSV sliders until green plants are cleanly selected
   - Press 'q' to save

### 3. Scheduled Captures

**Using cron (runs every 4 hours):**
```bash
crontab -e
```
Add:
```
0 */4 * * * cd /path/to/seedling-tracker/mac-mini && /usr/bin/python3 tracker.py --capture >> capture.log 2>&1
```

**Using launchd (macOS native):**

Create `~/Library/LaunchAgents/com.seedling.tracker.plist`:
```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
    <key>Label</key>
    <string>com.seedling.tracker</string>
    <key>ProgramArguments</key>
    <array>
        <string>/usr/bin/python3</string>
        <string>/path/to/seedling-tracker/mac-mini/tracker.py</string>
        <string>--capture</string>
    </array>
    <key>WorkingDirectory</key>
    <string>/path/to/seedling-tracker/mac-mini</string>
    <key>StartInterval</key>
    <integer>14400</integer>
    <key>StandardOutPath</key>
    <string>/path/to/seedling-tracker/mac-mini/capture.log</string>
    <key>StandardErrorPath</key>
    <string>/path/to/seedling-tracker/mac-mini/capture_err.log</string>
</dict>
</plist>
```
Load it:
```bash
launchctl load ~/Library/LaunchAgents/com.seedling.tracker.plist
```

## Usage

```bash
# First-time setup
python3 tracker.py --calibrate

# Manual capture
python3 tracker.py --capture

# View growth report with charts
python3 tracker.py --report

# Live preview (for aiming camera)
python3 tracker.py --preview
```

## Output Files

```
data/
├── images/              # Raw captures (timestamped)
├── annotated/           # Captures with grid overlay and measurements
├── growth_log.csv       # All measurements (cell_id, coverage %, timestamp)
├── growth_curves.png    # Growth chart (generated by --report)
└── heatmap.png          # Grid heatmap of latest readings
```

## Tips

- **Grow light color cast:** If using blurple/pink LEDs, widen the HSV hue range during calibration or briefly switch to white light for captures.
- **Condensation:** Keep camera above the humidity zone. A small USB fan near the lens helps.
- **Recalibrate** if you move the camera or tray — just run `--calibrate` again.
- **Yellowing detection:** Leaves shifting from green toward yellow will naturally show reduced coverage in the green HSV range — a dropping growth curve can indicate nutrient issues.
